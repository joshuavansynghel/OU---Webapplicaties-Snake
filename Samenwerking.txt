Hoe werken we samen in dit project. 

1. Clone de remote Git repo naar je eigen schijf
Voer onderstaande commmand uit in een directory
Command: git clone + <ssh of url>


2. Maak een eigen branch 

Nieuwe branch (in lokale repo)
Command: git branch <naam branch>

Switch naar branch
Switchen branch
Command: git checkout <naam branch> 

Maak en switch naar nieuwe branch
Command: git checkout  -b <naam branch>

Verwijder branch
Ga naar main branch
Command: git branch -D branch <naam branch>


3. Voer wijzigingen uit in eigen branch 
Command: git add . 
Command: git status 
Command: git commit -m "<message>" -m <toelichting> 


4. Push code naar remote Git repo (eigen branch)
Command: git push origin <naam branch> 


5. Creeer een pull request in Gitub 


6. Op gezamenlijke overleggen worden de commits doorgevoerd.
